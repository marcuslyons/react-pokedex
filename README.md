
# react-pokedex

A small Pok√©dex built using React for the purpose of using in presentations and other content.

## Run Locally

Clone the project

```bash
  git clone https://github.com/marcuslyons/react-pokedex
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn start
```

## Running Tests

To run tests, run the following command

```bash
  yarn test
```
